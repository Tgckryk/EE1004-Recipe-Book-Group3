# EE1004 - Recipe Book Loader (Group 3)

This is a Java-based Object-Oriented Programming project that loads recipes from a text file, organizes them,and allows users to list and filter them through a terminal interface. An optional Graphical User Interface (GUI) is also included for bonus evaluation.

## Requirements
* Java Development Kit (JDK) 25 or compatible.
* No external libraries are required (Java Swing is used for the GUI).

## How to Compile and Run
Navigate to the project folder via terminal/command prompt and run the following commands.

**To compile all files:**
```bash
javac *.java
```

To run the standard Terminal Version:
```bash
java Main
```

To run the Graphical User Interface (GUI) Version:
```bash
java GuiMain
```
